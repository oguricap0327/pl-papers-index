# Formal Verification

Papers on formal verification from POPL, PLDI, OOPSLA, ECOOP, and ICFP.

---

## 2026

- **[Formal Verification for JavaScript Regular Expressions](../venues/popl.md#formal-verification-javascript-regex)** — POPL 2026 🏆 Distinguished Paper  
  *Fully mechanized (Coq) semantics for all ECMAScript regex features.*

- **[Miri: Practical Undefined Behavior Detection for Rust](../venues/popl.md#miri)** — POPL 2026  
  *Interpreter-based runtime UB detection with a formalized memory model for Rust.*

## 2025

- **[Correctness Meets Performance: From Agda to Futhark](../venues/icfp.md#correctness-meets-performance-from-agda-to-futhark)** — ICFP 2025  
  *Bridge from Agda proofs to high-performance GPU code via formal correspondence.*

- **[Formal Semantics and Program Logics for a Fragment of OCaml](../venues/icfp.md#formal-semantics-and-program-logics-for-a-fragment-of-ocaml)** — ICFP 2025  
  *Mechanized separation logic for realistic OCaml fragment with higher-order functions.*

- **[McTT: A Verified Kernel for a Proof Assistant](../venues/icfp.md#mctt-a-verified-kernel-for-a-proof-assistant)** — ICFP 2025  
  *Verified kernel for a Martin-Löf type theory proof assistant, mechanized in Coq.*

- **[SecRef*: Securely Sharing Mutable References](../venues/icfp.md#secref-securely-sharing-mutable-references)** — ICFP 2025  
  *F* proof rules for secure sharing at verified/unverified code boundaries.*

- **[Verified Compilation of a Dependently Typed Language](../venues/pldi.md#verified-compilation)** — PLDI 2025  
  *Mechanized compiler correctness proof for a dependent type theory.*

- **[Verified Interpreters for Dynamic Languages with Applications to the Nix Expression Language](../venues/icfp.md#verified-interpreters-for-dynamic-languages-with-applications-to-the-nix-expression-language)** — ICFP 2025  
  *Iris/Coq-verified interpreter for Nix with lazy evaluation and dynamic typing.*

- **[Verifying Graph Algorithms in Separation Logic](../venues/icfp.md#verifying-graph-algorithms-in-separation-logic-a-case-for-an-algebraic-approach)** — ICFP 2025  
  *Algebraic approach to verifying graph algorithms in separation logic.*
